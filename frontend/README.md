# Traffic Monitoring Frontend

This is the frontend dashboard for the Traffic Monitoring System.

## Deployment to Vercel

1. Go to [vercel.com](https://vercel.com)
2. Sign up/Login with GitHub
3. Import this repository
4. Set the **Root Directory** to `frontend`
5. Click Deploy

## Configuration

After deploying the backend to Render, update the `API_BASE_URL` in `dashboard.html` with your Render URL.
